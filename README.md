# KUNST Launcher

A lightweight Flutter-based Android launcher for intentional planning, focus mode, and daily execution.

## Current status

This repository currently contains the initial project scaffold based on the KUNST specification, including:
- a Flutter app entrypoint
- a lightweight database layer for tasks and settings
- an onboarding screen for first launch

## Next steps

- add the remaining launcher screens from the specification
- wire Android permissions and platform channels
- implement focus-mode behavior and calendar integration
