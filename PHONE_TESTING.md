# KUNST Launcher Phone Test Checklist

## Before Install
1. Build a release APK from the workspace.
2. Install the APK on the phone.
3. Set KUNST Launcher as the default Home app.

## First Launch
1. Open the app.
2. Walk through the permission screen.
3. Grant device admin.
4. Allow calendar permissions.
5. Open notification listener settings.
6. Open usage access settings.
7. Open exact alarm settings.
8. Open battery optimization settings.

## Verify Core Flow
1. Confirm onboarding completes into Focus Home.
2. Add a task in Focus Home.
3. Change the task world from the dropdown.
4. Open Night Planning and save a note.
5. Change the night-planning world from the dropdown.
6. Open Evening Review and confirm it shows today's tasks.

## If Anything Fails
1. Check whether Android denied one of the permissions.
2. Confirm KUNST Launcher is still set as the Home app.
3. Reopen the permission screen from a fresh install if needed.
